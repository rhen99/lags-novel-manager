A plugin for managing novels and chapters in WordPress.

== Description ==
LAGS Novel Manager allows authors to structure their writing into novels and chapters with a clean frontend experience.

== Installation ==
1. Upload to /wp-content/plugins/
2. Activate the plugin
3. Use the new Novel and Chapter post types
