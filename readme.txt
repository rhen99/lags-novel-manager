A custom plugin to manage and read novels with a built-in chapter editor.

== Description ==

LAGS Novel Manager is a custom WordPress plugin that allows you to create, manage, and read novels with a structured chapter system.

It includes a custom admin editor for writing and organizing chapters efficiently.

== Features ==

= Core =
* Custom Post Types: Novels and Chapters
* Chapter-to-novel relationship
* Chapter numbering system

= Frontend =
* Novel page with chapter list
* Chapter reading page
* Next / Previous navigation
* Custom templates

= Admin Editor =
* Select and manage novels
* WYSIWYG chapter editor
* Load and edit chapters instantly

= Advanced Features =
* Autosave (interval-based)
* Create new chapters instantly
* Edit title and content
* Delete chapters
* Drag & drop chapter reordering
* Real-time UI updates

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/`
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Start creating novels and chapters

== Frequently Asked Questions ==

= Can I use this with any theme? =
Yes, but styling may vary depending on your theme.

= Does it support Gutenberg? =
The custom editor uses WordPress's classic editor (TinyMCE).
